# Restaurant Reservation System

## Description
System for managing restaurant reservations:
- Online booking (Web)
- Walk-in booking (WinForms)
- Backend API (Spring Boot)

## Project Structure
- backend/
- web-client/
- desktop-client/
- docs/

## Tech Stack
- Backend: Java Spring Boot
- Web: HTML/CSS/JS
- Desktop: C# WinForms
- Database: (MySQL / SQL Server)
